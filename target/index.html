    Target Interaction

    Another exciting way to create interaction in Aframe is by targeting other object when event is triggered from an object. For instance, you create two boxes in the scene. Both boxes are colored red. When you trigger the event on Box 1 however, Box 2 will change its color.

    This is particularly useful if you want to create interactive buttons that show or hide messages.

    Let's start with a simple example. Add one more boxes in the scene. Lets change its position attribute to 5 0 -5.

    <html>
  <head>
    <script src="https://aframe.io/releases/1.3.0/aframe.min.js"></script>
    <script src="https://unpkg.com/aframe-event-set-component@3.0.3/dist/aframe-event-set-component.min.js"></script>
  </head>

  <body>
    <a-scene>
      <a-box position="0 0 -5" color="red" event-set__color="_event: mouseenter; color: green"></a-box>
      <a-box position="5 0 -5" color="red"></a-box>

      <a-camera>
        <a-cursor></a-cursor>
      </a-camera>

    </a-scene>
  </body>
</html>

    Before we can add target interaction, we need to add id to give unique identifier to the object. Let's give Box 1 an id of "box1" and Box 2 an id of "box2".

    <html>
  <head>
    <script src="https://aframe.io/releases/1.3.0/aframe.min.js"></script>
    <script src="https://unpkg.com/aframe-event-set-component@3.0.3/dist/aframe-event-set-component.min.js"></script>
  </head>

  <body>
    <a-scene>
      <a-box id="box1" position="0 0 -5" color="red" event-set__color="_event: mouseenter; color: green"></a-box>
      <a-box id="box2" position="5 0 -5" color="red"></a-box>

      <a-camera>
        <a-cursor></a-cursor>
      </a-camera>

    </a-scene>
  </body>
</html>

    Next, under event-set__color attribute, we add _target: #box2 to target Box 2 for this interaction. Don't forget to include hashtag before the id name.

    <html>
  <head>
    <script src="https://aframe.io/releases/1.3.0/aframe.min.js"></script>
    <script src="https://unpkg.com/aframe-event-set-component@3.0.3/dist/aframe-event-set-component.min.js"></script>
  </head>

  <body>
    <a-scene>
      <a-box id="box1" position="0 0 -5" color="red" event-set__color="_event: mouseenter; color: green; _target: #box2"></a-box>
      <a-box id="box2" position="5 0 -5" color="red"></a-box>

      <a-camera>
        <a-cursor></a-cursor>
      </a-camera>

    </a-scene>
  </body>
</html>

    If we run the page in browser, we will see that Box 2 will change color upon cursor interaction with Box 1.

    There are several other properties that you can change with event-set component. Generally, every Primitives properties can be used for event-set interaction. Some properties along with their description is written in the table below:

    Properties Description
    position change the position of the object
    rotation change the rotation of the object
    visibility change the visibility of the object, to show or hide objects
    color change the color of the material
